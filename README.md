# Clean Home Pros LLC

Marketing site for Clean Home Pros LLC, converted from a Figma Make export
(Vite + React + react-router) to **Next.js 15 (App Router) + Tailwind CSS v4 + TypeScript**.

## Getting started

```bash
npm install
npm run dev      # http://localhost:3000
```

```bash
npm run build    # production build
npm start        # serve the production build
```

## Stack

- **Next.js 15** — App Router, React Server Components by default
- **React 19**
- **Tailwind CSS v4** — via `@tailwindcss/postcss`, tokens in `app/globals.css`
- **TypeScript** (strict)
- **lucide-react** — icons
- **next/font/google** — Plus Jakarta Sans + Lora, self-hosted at build time
